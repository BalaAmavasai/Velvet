Velvet

Vim Element Layout View Editor Tool (Velvet)

A modal, Vim-inspired code editor that runs entirely in your browser.

Quick Start

Open velvet, press i to enter Insert mode and start typing. Press Esc to return to Normal mode. Use : to run commands. Press / to search.

Modes

Velvet is a modal editor. Every key has a different meaning depending on the active mode.

Operators NORMAL

Operators combine with motions or text objects to act on a range of text. Doubling the operator key (e.g. dd) applies it to the current line.

OperatorActionExample
dDelete (cut)dw — delete word
cChange (delete + insert)c$ — change to end of line
yYank (copy)yy — yank current line
>Indent right>> — indent current line
<Indent left<< — de-indent current line
Composing: d3j deletes the current line plus 3 lines below. ci{ changes everything inside the nearest {}.

Text Objects NORMAL

Ex Commands COMMAND

Enter Command mode with :. Use ↑ / ↓ to browse command history. Press Enter to execute, Esc to cancel.

File

CommandAction
:wSave current buffer (opens system Save dialog on first save)
:w {name}Save as a new filename
:waSave all open buffers
:wq / :xSave and close buffer
:qClose buffer (fails if unsaved changes)
:q!Force close buffer, discarding changes
:eOpen file picker to load a file
:e {name}Create a new empty buffer with the given name

Navigation

CommandAction
:{N}Go to line N
:nohClear search highlights

Substitution COMMAND

:s/{pattern}/{replacement}/[flags]    — substitute on current line
:%s/{pattern}/{replacement}/[flags]   — substitute in entire file
FlagMeaning
gReplace all occurrences on the line (not just first)
iCase-insensitive match
Example: :%s/console\.log/logger.debug/g replaces every console.log in the file.

Line-range delete

:{start},{end}d    — delete lines from start to end (e.g. :3,7d)

CommandEffect

File I/O

Velvet uses the File System Access API where available (Chrome, Edge). Other browsers fall back to file download (save) and a file input picker (open).

Each open file is a buffer. Buffers are shown as tabs at the top of the editor. Click a tab to switch, or use commands.

Ctrl→Next buffer
Ctrl←Previous buffer

Macros NORMAL

Record a sequence of keystrokes into a register and replay them.

KeyAction
q{a-z}Start recording macro into register a–z
qStop recording (when a macro is active)
@{a-z}Play back macro from register a–z
Example: qa → make edits → q → 10@a replays the macro 10 times.

Registers NORMAL

Named registers store text independently. By default all operations use the unnamed register ".

KeyAction
"{a-z}Select register for the next yank, delete, or paste
Example: "ayy yanks the current line into register a. Then "ap pastes it.

Auto-pair & Smart Typing INSERT

Velvet automatically inserts matching closing characters and skips over them when you type them.

TypeResult
(Inserts (), cursor between
[Inserts [], cursor between
{Inserts {}, cursor between
"Inserts "", cursor between
'Inserts '', cursor between
`Inserts ``, cursor between
Type closing char when already presentMoves cursor past it (no duplicate)

Auto-indent is applied on Enter: the new line inherits the indentation of the previous line, with an extra level after { or :.
